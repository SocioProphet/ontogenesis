# Examples

This folder contains **instance graphs** and JSON-LD documents used to test:

- SHACL promotion gates
- JSON-LD contexts
- Inter-module linking (Prophet ⇄ EPI ⇄ Noether ⇄ Domains)

Files:
- `layer_contract_demo.ttl` — example Noetherian layer contract (SU(2), ℍ)
- `capd_demo.jsonld` — example Prophet capability descriptor
- `quantum_lane_demo.jsonld` — quantum lane (opt-in=false) example
- `person_demo.ttl` — person + phenotype observation example
- `michael-belief-state-demo.jsonld` — Michael belief-state example
- `michael-candidate-law-demo.jsonld` — Michael candidate-law example
- `human-digital-twin-demo.jsonld` — bounded human digital twin example
- `valueflows-governed-task-flow-demo.jsonld` — compact governed task-flow example showing completed, revoked, and canceled terminal states across process run, delegation, capability grant, task, commitment, and checkpoint state
