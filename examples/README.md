# Examples

This folder contains **instance graphs** and JSON-LD documents used to test:

- SHACL promotion gates
- JSON-LD contexts
- Inter-module linking (Prophet ⇄ EPI ⇄ Noether ⇄ Domains)

Files:
- `layer_contract_demo.ttl` — example Noetherian layer contract (SU(2), ℍ)
- `capd_demo.jsonld` — example Prophet capability descriptor
- `quantum_lane_demo.jsonld` — quantum lane (opt-in=false) example
- `person_demo.ttl` — person + phenotype observation example
