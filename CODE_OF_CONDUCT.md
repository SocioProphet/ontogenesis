# Code of Conduct

## Our commitment

We are committed to providing a welcoming, respectful, and harassment-free experience for everyone participating in this project.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and focus on technical outcomes.
- Give and receive feedback professionally.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Trolling, intimidation, or sustained disruption.
- Sharing private information without consent.

## Scope

This Code of Conduct applies to project spaces, issues, pull requests, discussions, and other community interactions related to this repository.

## Reporting

For conduct concerns, report privately via the contacts listed in [`SUPPORT.md`](SUPPORT.md).

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this Code of Conduct.
